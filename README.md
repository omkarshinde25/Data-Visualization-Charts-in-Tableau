# Types of Charts in Tableau  
A complete guide to commonly used chart types in Tableau, with definitions, use cases, and when to apply each chart.  
Created by **Omkar Shinde**.

---

## 1) Bar Charts  

### A) Row Bar Chart (Horizontal Bar Chart)
<img src="https://github.com/omkarshinde25/Data-Visualization-Charts-in-Tableau/blob/main/Chart%20Photos/Rows%20Bar%20Chart.png" width="800">

- Used to compare categories clearly using horizontal bars  
- Best when category names are long  
- Simple and easy-to-read layout  
- Useful for ranking items from highest to lowest because horizontal layout makes sorting clear  
- Ideal when the viewer needs to read long labels without text overlapping  

---

### B) Column Bar Chart (Vertical Bar Chart)
<img src="https://github.com/omkarshinde25/Data-Visualization-Charts-in-Tableau/blob/main/Chart%20Photos/Columns%20Bar%20Chart.png" width="800">

- Used to compare values across categories  
- Helps analyze differences or trends  
- Ideal for showing changes over time when categories are few  
- Useful for dashboards because vertical bars are visually intuitive and easy to interpret  

---

### C) Side-by-Side Bar Chart
<img src="https://github.com/omkarshinde25/Data-Visualization-Charts-in-Tableau/blob/main/Chart%20Photos/Side%20by%20Side%20Bar%20Chart.png" width="800">

- Compares multiple measures or sub-categories  
- Best for detailed comparison  
- Useful for male vs female or budget vs actual comparisons  
- Great for spotting differences between subcategories inside the same group  

---

### D) Time-Based Bar Chart
<img src="https://github.com/omkarshinde25/Data-Visualization-Charts-in-Tableau/blob/main/Chart%20Photos/Time%20Based%20Bar%20Chart.png" width="800">

- Displays data over time (day, month, year)  
- Helps identify seasonality and trends  
- Useful when you want to analyze month-wise or year-wise performance  
- Helps quickly compare time periods to understand growth or decline  

---

### E) Stacked Bar Chart
<img src="https://github.com/omkarshinde25/Data-Visualization-Charts-in-Tableau/blob/main/Chart%20Photos/Stacked%20Bar%20Chart.png" width="800">

- Shows total value + contribution of each category  
- Great for part-to-whole comparison  
- Useful for tracking category contributions across time  
- Ideal when you want to view composition without losing total value information  

---

### F) Full Stacked Bar Chart (100% Stacked Bar)
<img src="https://github.com/omkarshinde25/Data-Visualization-Charts-in-Tableau/blob/main/Chart%20Photos/Full%20Stacked%20Bar%20Chart%20.png" width="800">

- Shows percentage contribution  
- Useful for comparing proportions across groups  
- Helps understand distribution patterns even when actual values are not important  
- Great for comparing relative performance between categories  

---

### G) Multiple Bar Chart
<img src="https://github.com/omkarshinde25/Data-Visualization-Charts-in-Tableau/blob/main/Chart%20Photos/Multiple%20Bar%20Chart.png" width="800">

- Displays several bar charts in one view  
- Best for comparing multiple measures  
- Useful when you want to analyze different KPIs without mixing data  
- Helps avoid clutter by separating related charts cleanly  

---

### H) Bar-in-Bar Chart
<img src="https://github.com/omkarshinde25/Data-Visualization-Charts-in-Tableau/blob/main/Chart%20Photos/Bar%20in%20Bar%20Chart.png" width="800">

- Compares two related measures inside the same bar  
- Example: Actual vs Target  
- Ideal for showing performance gaps clearly  
- Useful when comparing two metrics with minimal space  

---

### I) Bar Code Chart
<img src="https://github.com/omkarshinde25/Data-Visualization-Charts-in-Tableau/blob/main/Chart%20Photos/Bar%20Code%20Chart.png" width="800">

- Shows data distribution using thin bars  
- Helps identify patterns & gaps  
- Useful for spotting clusters and outliers in data  
- Great for high-volume datasets without cluttering visual  

---

## 2) Line Charts  

### A) Time-Based Line Chart
<img src="https://github.com/omkarshinde25/Data-Visualization-Charts-in-Tableau/blob/main/Chart%20Photos/Times%20based%20Line%20Chart.png" width="800">

- Shows trends over time  
- Great for patterns and seasonality  
- Useful for forecasting future values  
- Helps identify time-series anomalies (spikes, dips)  

---

### B) Customized Line Chart
<img src="https://github.com/omkarshinde25/Data-Visualization-Charts-in-Tableau/blob/main/Chart%20Photos/Customized%20Line%20Chart.png" width="800">

- Modified colors, thickness, markers  
- Highlights specific insights  
- Useful when presenting data to highlight main trends  
- Helps create branded or visually enhanced dashboards  

---

### C) Multiple Line Chart
<img src="https://github.com/omkarshinde25/Data-Visualization-Charts-in-Tableau/blob/main/Chart%20Photos/Multiple%20Line%20Chart.png" width="800">

- Compares multiple categories or measures  
- Shows relationships and differences  
- Useful for comparing sales of multiple regions or products  
- Helps spot how different groups behave over the same timeline  

---

### D) Dual Line Chart
<img src="https://github.com/omkarshinde25/Data-Visualization-Charts-in-Tableau/blob/main/Chart%20Photos/Dual%20Line%20Chart.png" width="800">

- Two different measures with dual axes  
- Helps compare metrics with different scales  
- Useful when measures do not share the same range (revenue vs quantity)  
- Ideal for analyzing correlation between two different KPIs  

---

### E) Cumulative Line Chart
<img src="https://github.com/omkarshinde25/Data-Visualization-Charts-in-Tableau/blob/main/Chart%20Photos/Cumulative%20Line%20Chart.png" width="800">

- Displays running totals over time  
- Useful for tracking growth  
- Ideal for year-to-date or month-to-date progression  
- Helps visualize how values accumulate across a timeline  

---

### F) Multiple Line Chart with Color
<img src="https://github.com/omkarshinde25/Data-Visualization-Charts-in-Tableau/blob/main/Chart%20Photos/Multiple%20Line%20Chart.png" width="800">

- Shows several colored lines for clarity  
- Makes comparison easier  
- Very useful when comparing multiple groups at once  
- Colors help quickly differentiate categories  

---

### G) Highlighted Line Chart
<img src="https://github.com/omkarshinde25/Data-Visualization-Charts-in-Tableau/blob/main/Chart%20Photos/Highlighted%20Line%20Chart.png" width="800">

- Emphasizes one category while others fade  
- Best for storytelling and presentation  
- Useful when drawing attention to one specific trend  
- Helps maintain context while keeping focus on the primary line
